I am a lecturer (Assistant Professor) in Faculty of Information Technology, Monash University, Australia. I obtained my PhD from School of Computer Science and Engineering, Nanyang Technological University (NTU), Singapore. When pronouncing my first name, "Chun-yang" sounds like "train-young". My research focuses on Software Engineering, Deep Learning and Human-Computer Interaction. Specifically, I apply AI/ML, NLP and program analysis technology in the following three directions:

  • AI-assisted Automated Mobile App Development:
    e.g., How to generate front-end code for the UI design? (ICSE’18)
    e.g., How to help designer get inspiration for UI design? (CSCW’20)
    e.g., Can mobile apps more accessible for people with visual impairments? (ICSE’20)
  • (AI-empowered) Mining Software Repository:
    e.g., How to extract word deviations from unstructured text? (ICSE’17)
    e.g., Can manual collaborative editing in Q&A posts be automated by deep learning? (CSCW’17)
    e.g., How to distil the difference between similar software technologies? (ASE’18)
  • Testing Deep Neural Network and Mobile App Security:
    e.g., How to ensure the robustness of neural network? (ASE’18)
    e.g., How to generate the phishing apps by deep learning? (TIFS’19)
